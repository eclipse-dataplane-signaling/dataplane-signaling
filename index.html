Data Plane Signaling 1.0-RC2

A specification for interoperable communication between Dataspace Protocol control planes and data planes.

Status of this document

This version (1.0-RC2) of the Data Plane Signaling specification is a release of the specification and considered to be stable. Further changes shall not affect conformity. All changes made to the specification can be reviewed in the GitHub repository. This specification is affiliated with the Eclipse Dataspace Working Group.
